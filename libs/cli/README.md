# ZTC CLI

Terminal interface for ZeroTouch Composition Engine workflows.

## Features

- Core lifecycle commands (init, render, validate, bootstrap, eject)
- Workflow management commands
- Rich terminal UI
- Interactive prompts

## Installation

```bash
pip install ztc-cli
```

## Usage

```bash
# Workflow commands
ztc-workflow list
ztc-workflow run <workflow-name>

# Core commands (via main ztc CLI)
ztc init
ztc render
ztc validate
```
