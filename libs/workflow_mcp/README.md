# ZTC Workflow MCP Server

Model Context Protocol server integration for ZTC workflows.

## Features

- MCP protocol implementation
- Workflow observation and monitoring
- Transport security modes
- Storage integration

## Installation

```bash
pip install ztc-workflow-mcp
```

## Usage

```python
from workflow_mcp import WorkflowMCPServer

server = WorkflowMCPServer()
# Configure and run server
```
